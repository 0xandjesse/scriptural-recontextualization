# Nag Hammadi Library - Master Index

## Overview
The Nag Hammadi Library contains 13 codices (books) of Gnostic texts discovered in 1945 near Nag Hammadi, Egypt. This collection represents some of the earliest Christian Gnostic writings, offering alternative perspectives on early Christian thought.

**Source**: Gnosis.org Nag Hammadi Library Codex Index - Complete scholarly reference
